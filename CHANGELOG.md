## [0.1.0] - 2024-12-24

### Added

- Initial release
- Rust code analysis using Ollama LLMs
- LLM-driven mutation testing
- Web dashboard for configuration and results viewing
- Idle detection and scheduled analysis
- Multi-endpoint Ollama support
- SQLite database for persistent storage
- Repository management UI
- File analysis results viewer
- Mutation testing results viewer

### Known Limitations

- Currently supports Rust codebases only
- Requires Ollama to be installed
- Linux and macOS only (no Windows support yet)
